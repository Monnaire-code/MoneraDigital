## 部署详情

### 本次变更说明

**问题修复**:
- **B-1**: 修复 GetStats 缺少 singleflight，防止并发请求重复查询数据库
- **B-2**: 修复缓存返回指针导致的潜在数据污染问题

**技术方案**:
- 使用 `golang.org/x/sync/singleflight` 合并并发请求
- 添加深拷贝函数 `cloneFundStatsData` 隔离缓存数据

## 回滚流程

### 快速回滚

```bash
# SSH 到服务器
ssh ec2-user@<production-server>

cd /home/ec2-user/monera

# 停止服务
sudo systemctl stop monera-digital

# 恢复备份
cp server.bak server

# 启动服务
sudo systemctl start monera-digital

# 验证
curl http://localhost:8081/api/health
```
